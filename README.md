# 🏁 CERO — El Coche de Internet (Master Plan v16.0)

> **El primer monoplaza de calle desarrollado 100% en abierto por internet, partiendo de 0€ y construido mediante el poder de la comunidad.**

---

## 🎯 Visión y Alcance
CERO es un experimento social, creativo y de ingeniería de código abierto para fabricar un vehículo monoplaza eléctrico homologable en España (ITV individual) en 28 meses.

Este repositorio actúa como el **Data Room maestro** y centro de operaciones del proyecto. Contiene toda la documentación estratégica, operativa, de reclutamiento, de marketing y de sourcing necesaria para coordinar el trabajo descentralizado del Core Team y los colaboradores.

---

## 📂 Estructura del Repositorio

El repositorio está estructurado en 6 carpetas operativas conteniendo **31 documentos maestros** generados en formatos PDF y Markdown (dentro de `documents/`):
